# Code Reviews

This directory contains code review artifacts generated by the Reviewer Agent.

## File Format

Each review is saved as `REVIEW-{task_id}.md` with:
- Machine-readable metadata in HTML comment
- Status: PASS or FAIL
- Spec alignment check
- Domain rules compliance
- Security scan results
- Verdict and fix instructions (if FAIL)

## Usage

```
/review <task_id>   # View review status
/rules              # View domain rules
```

# QA Reports Directory

This folder contains quality assurance artifacts generated by the Tribunal system.

## Contents
- **Tribunal Reports:** Automated code review findings
- **Audit Logs:** Security and compliance checks
- **Test Coverage:** Coverage reports and analysis

## Usage
Reports are generated automatically by `/verify` and `/ship` commands.
Manual reviews can be triggered via `/review <scope>`.

## Note
This folder is tracked in git. Sensitive findings should be redacted before commit.
